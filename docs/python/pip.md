
linux下指定位置为

```
$HOME/.config/pip/pip.conf
或者
$HOME/.pip/pip.conf
```

mac下指定位置为
```
$HOME/Library/Application Support/pip/pip.conf
或者
$HOME/.pip/pip.conf
```

windows下指定位置为
```
%APPDATA%\pip\pip.ini
或者
%HOME%\pip\pip.ini
```
