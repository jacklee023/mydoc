1.1 pickle

1.1.1 dump的参数

pickle.dump(obj, file, protocol=None, *, fix_imports=True)

pickle.dump的protocol参数可以指定序列化模式，其中0表示文本文件，1以上表示二进制文件 配合fix_imports参数可以使序列化文件可以兼容python2

1.2 mccabd

1.2.1 介绍

Cyclomatic Complexity 圈复杂度 McCabe 是一种度量程序复杂度的方法，如果单个子程序复杂度过高，或许就需要拆分逻辑提高程序的易读性。

  • 圈复杂度和McCabe

1.2.2 示例

>> pip install mccabe
>> python -m mccabe factorial.py 
>> python -m mccabe --min 5 mccabe.py
>> python -m mccabe factorial.py -d  # 产生dot有向图
  • 产生dot有向图
